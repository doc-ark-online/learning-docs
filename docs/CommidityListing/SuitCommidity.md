# 组合商品类

1. **套装**
   - 套装需至少包括上衣、下衣、头发、鞋4个部分或者连体衣、头发、鞋3个部分的组合。
2. **全妆**
   - 全妆需至少包括眉毛，睫毛，瞳孔样式，眼影，腮红, 口红中的任意5个部位的组合。

3. **注意事项**

- 套装、全妆中使用到的商品，均已过审上线至游戏商城。
- 单件商品不可重复组合。
- 组合商品价格默认为单件总和，不可二次定价。
- 组合商品中使用到的部件需由单个创作者账号制作和上传，多个账号之间上传的商品不可组合。
- 组合商品必须制作UI图标。

4. **组合商品发布流程**

- 勾选商品页面需要组合的商品——点击新建组合商品——填写组合商品信息——点击提交

# 如何在口袋方舟中预览特殊形象

:::tip **前言**

在上传特殊形象/皮套人资源的时候，当前的资源导入工具暂时不支持预览动态的特殊形象资源。 下文会介绍如何使用口袋方舟的功能来实现在不上传资源的情况下预览带动作的特殊形象资源。

:::



#### 步骤

1. 将特殊形象资源导入至口袋方舟。

<img src="https://arkimg.ark.online/1777372332027-6.webp" alt="img" style="zoom:80%;" />

2. 在【对象管理器】中找到Player，点击后在属性面板中找到【使用233平台形象】选项，将其设置为关闭。

<img src="https://arkimg.ark.online/1777372332025-1.webp" alt="img" style="zoom:80%;" />

3. 在属性面板中将【外观类型】修改为“基础人形形象”。

<img src="https://arkimg.ark.online/1777372332026-2.webp" alt="img" style="zoom: 50%;" />

4. 将导入好的资源拖动到【整体形象】栏中

<img src="https://arkimg.ark.online/1777372332026-3.webp" alt="img" style="zoom: 50%;" />

5. 点击运行按钮，即可在窗口中预览资源穿戴时的效果

<img src="https://arkimg.ark.online/1777372332026-4.webp" alt="img" style="zoom: 80%;" />

6. 可以点击跳跃、下蹲等按钮，测试资源在做动作时的效果。

<img src="https://arkimg.ark.online/1777372332026-5.webp" alt="img" style="zoom: 67%;" />
